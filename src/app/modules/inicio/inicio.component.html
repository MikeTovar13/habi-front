Bienvenido a App Habi

Aquí podrás encontrar Inmuebles junto con sus propietarios, las funcionalidades iniciales de esta aplicación son:

  1. Propietarios
    Iinicia creando un propietario para después crear inmuebles, cuando eliminas un propietario también eliminarás todos sus inmuebles.
    V 1.0
  2. Inmuebles
    Puedes crear inmuebles desde cada "Card" de propietario, puedes crear uno o muchos según lo desees, después de creados puedes verlos en el menú de "Inmuebles" y también eliminarlos.
    V 1.0
  3. Paises
    Encontraras un listado de paises que se alimenta de la información obtenida a través de un servicio REST API gratuito. La documentación de uso esta aquí: REST COUNTRIES
    V 1.0
